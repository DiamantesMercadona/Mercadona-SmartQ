# Backend - Mercadona SmartQ

## Requisitos

- **Python 3.8+** instalado en el sistema o disponible en un entorno virtual (venv).

## Instalación y Despliegue

### 1. Instalar las dependencias

```bash
python -m pip install -r requirements.txt
```

### 2. Ejecutar el punto de entrada

```bash
python main.py
```

## Pruebas (Tests)

Para ejecutar todos los tests unitarios del backend, ejecutar el siguiente comando:

```bash
python -m unittest discover tests
```
